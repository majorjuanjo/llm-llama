# Estructura y Formato del Workshop

4 sesiones (1 presencial + 3 virtuales) → agente funcional.
S1: Estrategia y caso de uso
S2: Instalación y agente local
S3: RAG con documentos
S4: Producción y escalamiento

{% hint style="info" %}ℹ️ Notificación: El diseño secuencial evita instalar antes de validar caso (reduce abandono).{% endhint %}
{% hint style="success" %}✅ Recomendación: Captura KPI inicial (tiempo tarea manual) durante S1 para comparar en S4.{% endhint %}

> [!NOTE] Secuencia = estrategia → técnica → datos → producción.
> [!TIP] KPI inicial imprescindible para ROI.
