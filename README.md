# llm-llama

Taller y guía práctica para construir tu propio stack de IA con Llama (open‑source) orientado a MiPyMEs y startups LATAM. Reúne sesiones, playbooks, plantillas No‑Code, FAQ ampliado y materiales listos para producción.

Componentes clave
- Sesión 1: Estrategia y Caso de Uso (Canvas, Triángulo PyME, Mini AI Strategy Compass, Prompt v1.0)
- Sesión 2: Instalación y primer agente local (Ollama/LM Studio) + automatización (Make/Zapier/n8n)
- Sesión 3: RAG con documentos propios (citas y evaluación)
- Sesión 4: Producción, métricas, compliance y plan 30‑60‑90

Capas de robustecimiento (10)
1) Fundamentos (existente)
2) Sistema de métricas y progreso visible
3) Microlearning entre sesiones
4) Templates ejecutables (copy‑paste ready)
5) Certificación y badging
6) Comunidad activa y peer learning
7) Contenido multimedia rico
8) ROI Calculator y business case
9) Compliance y Legal Kit
10) Alumni Network y continuidad
